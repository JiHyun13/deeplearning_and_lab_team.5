기존: 텍스트에서 감정 단어 제거 시 모델이 문맥만으로 감정 분류할 수 있는가?
멀티모달 확장:  음성, 영상 추가 => 감정 단서 보완

: 감정 단어가 없는 상황에서 어떤 모달리티가 감정을 가장 잘 보완하는가?

ex) text / Audio / Video 성능을 각각 조절해본다
- 감정 단어 제거 : 음성 + 영상 + 감정 제거된 텍스트 => 비언어 정보가 감정 보완 가능한가
- 음성 제거 : 텍스트 + 영상  => 표정이 음성을 대체 가능한가
- 영상 제거 : 텍스트 + 음성 => 목소리만으로 충분한가
- 텍스트 제거 : 음성 + 영상  => 언어 없이 감정 추론 가능한가
=> 모델은 감정마다 어떤 정보를 더 신뢰하는가?
